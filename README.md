# xyshzs

This workspace contains a demonstration repository for a "campus life assistant" application, including a user-facing errand platform with order creation and a separate runner dashboard.

## camp_lassistant-app

The `camp_lassistant-app` folder holds a simple Node/Express backend plus static HTML/CSS pages for chat, orders, and forum features.  It is intended as a starting point for developing a more complete cross‑platform system.

See `camp_lassistant-app/README.md` for run instructions and details.
